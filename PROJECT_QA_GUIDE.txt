# SVM Satellite Image Classification - Project Q&A Guide
## Bachelor's Thesis Presentation Reference

---

## 1. WHAT IS THIS PROJECT ABOUT?

"This project applies Support Vector Machine (SVM), a supervised machine learning 
algorithm, for satellite image classification. I used the EuroSAT dataset which 
contains satellite images of 10 different land cover types like forests, rivers, 
highways, and residential areas."

## 4. WHERE DID YOU GET THE DATASET?

### EuroSAT (Satellite Images)
- Source: Zenodo (European Research Repository)
- URL: https://zenodo.org/record/7711810
- Citation: Helber et al., 2019
- Size: 27,000 images (64x64 pixels)
- Classes: 10 land cover types

### Land Cover Classes:
1. AnnualCrop - Annual crop fields
2. Forest - Forested areas
3. HerbaceousVegetation - Herbaceous vegetation
4. Highway - Roads and highways
5. Industrial - Industrial areas
6. Pasture - Pasture lands
7. PermanentCrop - Permanent crop fields
8. Residential - Residential areas
9. River - Rivers and streams
10. SeaLake - Seas and lakes

## 6. HOW DID YOU CREATE THE GRAPHS?

"The visualizations were created using Matplotlib for plotting and Seaborn
